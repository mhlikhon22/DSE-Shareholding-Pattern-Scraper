## Key Features

- Scrapes the shareholding pattern for every DSE-listed company  
- Automatically navigates through all company pages  
- Extracts structured data for:
  - Sponsors/Directors  
  - Government Holdings  
  - Institutions  
  - Foreign Investors  
  - General Shareholders  
- Cleans and organizes the data into a consistent tabular format  
- Exports the final output to Excel/CSV for seamless use in research, reporting, and analytics
